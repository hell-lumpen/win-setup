# Инструкция по запуску PowerShell скрипта

## Предварительные требования

1. **Запуск PowerShell с правами администратора**:
   - Откройте меню "Пуск".
   - Найдите "Windows PowerShell".
   - Щелкните правой кнопкой мыши на "Windows PowerShell" и выберите "Запуск от имени администратора".

2. **Убедитесь, что у вас установлен Chocolatey**:
   - Если не установлен, скрипт сам установит Chocolatey.

## Шаги для запуска скрипта

1. **Создайте файл скрипта**:
   - Откройте текстовый редактор (например, Notepad).
   - Вставьте скрипт PowerShell в новый файл.
   - Сохраните файл с расширением `.ps1`, например, `setup.ps1`.

2. **Откройте PowerShell с правами администратора**:
   - Если вы уже не сделали этого, откройте PowerShell как администратор.

3. **Разрешите выполнение скриптов**:
   - Выполните следующую команду для разрешения выполнения скриптов (в PowerShell с правами администратора):
     ```powershell
     Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
     ```

4. **Перейдите в каталог, где находится ваш скрипт**:
   - Используйте команду `cd` для перехода в нужный каталог. Например:
     ```powershell
     cd C:\Path\To\Your\Script
     ```

5. **Запустите скрипт**:
   - Выполните скрипт с помощью следующей команды:
     ```powershell
     .\setup.ps1
     ```

6. **Следите за выводом скрипта**:
   - Скрипт выполнит все необходимые действия и настроит вашу систему. Следите за выводом в консоли для получения информации о ходе выполнения и возможных ошибках.

## Дополнительные советы

- **Проверьте требования к системе**:
  - Убедитесь, что ваша система поддерживает WSL2 и все необходимые компоненты.

- **Обновите Windows**:
  - Перед запуском скрипта убедитесь, что ваша система обновлена до последней версии Windows 11.

- **Проверка после выполнения**:
  - После выполнения скрипта проверьте, что все приложения и настройки применены корректно.

Если у вас возникли вопросы или проблемы, обратитесь к документации PowerShell или поддержке вашей системы.
